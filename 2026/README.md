CAESAR Workshop Website
=======================
This is the website for the Workshop **CAESAR: Causal Anomalies - Exploring Synergies of Anomaly Detection and Causal Inference** held at ECML PKDD 2026.

The website is built for Google Pages 

Dependencies:
-------------
* Ruby >= 2.7
* RubyGems 
* GCC & Make
* Jekyll (see: https://jekyllrb.com/docs/installation/ubuntu/)
* Bundler

Local Testing
-------------
```bash
bundle install
bundle exec jekyll serve --trace --livereload
```
